## Bitwise Operator

* Bitwise Operators are used to perform on each binary munbers.These operators include bitwise AND, OR, XOR, and more.

 ## List of Bitwise Operator
```python
  Operator  Description    Example
      &     Bitwise AND   5 & 3 = 1
      |     Bitwise OR    5 | 3
      ^     Bitwise XOR   5 ^ 3 = 6
      ~     Bitwise NOT   ~5 = -6
      <<     Left shift   5 << 1 = 10
      >>    Right shift   5 >> 1 = 2
```
## Example

## Bitwise AND

```python
a = 5 
b = 3
result = a & b
```

## Bitwise OR
```python
a = 3
b = 2
result = a | b
```
